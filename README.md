# Daily Market Briefing

Automated pre-open market rundown, published every weekday at ~8:30 AM ET.

- **Live dashboard:** https://StackinChedduh.github.io/daily-market-briefing/
- Daily markdown archive in [`rundowns/`](./rundowns)

Generated by a scheduled Claude task running locally on the author's machine.
Each weekday morning the task reads market news, Discord channels, and data
sources, produces a markdown briefing, updates a single rolling HTML
dashboard, and pushes the result here.
